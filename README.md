# Data-maining and Time Series Analysis
Zawiera zadania, raporty oraz projekty z eksploracji danych realizowane w języku R. Repozytorium obejmuje techniki klasyfikacji, klasteryzacji, redukcji wymiarów oraz analizę dużych zbiorów danych. Dodatkowa są zawarte sprawozdania wraz z kodami z analizy szeregów czasowych (repozytorium: "listy z rozwiązaniami") zawierające zasotoswanie metody testów białoszumowości, dekompozycji, różnicowania, prognozowania (modele ARIMA) itp.
